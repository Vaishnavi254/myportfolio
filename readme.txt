## Features and Functionality

### Navigation

The website includes smooth scrolling between different sections, and the navigation menu dynamically highlights the active section based on the user's scroll position.

### Back to Top Button

A floating "Back to Top" button appears when the user scrolls down the page. When clicked, it smoothly scrolls the page back to the top for ease of navigation.

### Responsive Design

The portfolio is fully responsive, meaning it adapts to different screen sizes and devices, such as smartphones, tablets, and desktops.

## Installation

If you want to run the portfolio on your local machine, follow these steps:

### Prerequisites

Make sure you have the following installed:

- A modern web browser (Google Chrome, Firefox, etc.)
- A code editor (VS Code, Sublime Text, etc.)

### Steps

1. Clone the repository to your local machine:
    ```bash
    git clone https://github.com/Vaishnavi254/myportfolio.git
    ```

2. Open the project folder in your code editor.

3. Open the `index.html` file in your browser to view the portfolio.
4. My website is live at https://vaishnavi254.github.io/myportfolio/
